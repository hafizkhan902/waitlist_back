OAuth Callback Demo

Google OAuth Integration

This demo shows how to handle OAuth callbacks in your frontend.

Frontend Implementation Guide

1. OAuth Callback Handling

Your frontend should handle these URLs:

/auth/success?data=... // Successful OAuth callback /auth/error?data=... // Failed OAuth callback

2. React Implementation Example

import { useEffect } from 'react'; import { useSearchParams, useNavigate } from 'react-router-dom'; function AuthCallback() { const [searchParams] = useSearchParams(); const navigate = useNavigate(); useEffect(() => { const dataParam = searchParams.get('data'); if (dataParam) { try { const data = JSON.parse(decodeURIComponent(dataParam)); if (data.success && data.user) { // Store user data in state/context localStorage.setItem('user', JSON.stringify(data.user)); // Redirect to dashboard or success page navigate('/dashboard'); } else { // Handle error navigate('/auth/error', { state: { error: data.error } }); } } catch (error) { console.error('Error parsing OAuth data:', error); navigate('/auth/error'); } } }, [searchParams, navigate]); return <div>Processing authentication...</div>; }

3. Next.js Implementation Example

import { useEffect } from 'react'; import { useRouter } from 'next/router'; export default function AuthCallback() { const router = useRouter(); useEffect(() => { const { data } = router.query; if (data) { try { const parsedData = JSON.parse(decodeURIComponent(data)); if (parsedData.success && parsedData.user) { // Store user data localStorage.setItem('user', JSON.stringify(parsedData.user)); // Redirect to dashboard router.push('/dashboard'); } else { router.push('/auth/error'); } } catch (error) { console.error('Error parsing OAuth data:', error); router.push('/auth/error'); } } }, [router.query]); return <div>Processing authentication...</div>; }

4. Environment Variables

Make sure to set the FRONTEND_URL in your .env file:

FRONTEND_URL=http://localhost:3000
